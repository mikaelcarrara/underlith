# Roadmap

Underlith's strategic milestones for 2026.

## 2026 Plan

- **Q1 2026 — Developer Experience**
  - Documentation site improvements: Table of Contents, Prism highlighting, badge standardization.
  [x] Motion tokens: durations, easings, delays and composite tokens; reduced-motion policy.
  [x] CSS consumption examples across plain CSS and CSS-in-JS.
  - JSON Schema draft for canonical tokens.

- **Q2 2026 — Automation & Artifacts**
  - Optional generators for `underlith.json` and `underlith.d.ts`.
  - Token-aware linters and hardcoded-value detection.
  - CI templates for breaking-change detection and audits.

- **Q3 2026 — Adapters & Theming**
  - Theming guidance and alias strategy.
  - Additional composite motion presets.
  - Starter repos for common stacks (vanilla, React).

- **Q4 2026 — Hardening**
  - Visual regression harness for reference components.
  - Governance dashboards and audit trail surfacing.
