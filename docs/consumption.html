Consumption

Underlith defines what. Consumers decide how.

CSS consumption examples across plain CSS and CSS-in-JS.

Plain CSS

.card {
  padding: var(--space-4);
  background: var(--color-surface);
  color: var(--color-text-primary);
  border-radius: var(--radius-md);
}

Tailwind (v4)

Map tokens into Tailwind’s theme layer using @theme. Utilities will reference var(--token) directly. 

/* app.css */
@import "tailwindcss";

@theme {
  /* Colors */
  --color-surface: var(--color-surface);
  --color-brand: var(--color-brand-primary);
  --color-brand-hover: var(--color-brand-hover);
  --color-foreground: var(--color-text-primary);
  --color-inverse: var(--color-text-inverse);

  /* Spacing */
  --spacing-2: var(--space-2);
  --spacing-4: var(--space-4);

  /* Radii */
  --radius-sm: var(--radius-sm);
  --radius-md: var(--radius-md);

  /* Elevation */
  --shadow-sm: var(--shadow-sm);
}
<div class="rounded-md p-4 bg-surface text-foreground shadow-sm">
  <h3 class="text-lg font-semibold mb-2">Card</h3>
  <p>Powered by tokens via Tailwind v4.</p>
  <button class="mt-2 px-4 py-2 rounded-sm bg-brand text-inverse hover:bg-brand-hover">
    Action
  </button>
</div>

CSS Preprocessors (Sass / Less)

Preprocessors do not transform tokens. They only organize consumption. 

.card {
  padding: var(--space-4);
  background: var(--color-surface);
  border-radius: var(--radius-md);
}

CSS-in-JS

// styled-components / emotion
const Card = styled.div`
  padding: var(--space-4);
  background: var(--color-surface);
  border-radius: var(--radius-md);
`;
// vanilla-extract
export const card = style({
  padding: 'var(--space-4)',
  background: 'var(--color-surface)',
  borderRadius: 'var(--radius-md)',
});

AI / Automation

Generate a Card component using only:
- space-4
- color-surface
- radius-md

Do not introduce hardcoded values.
Output plain CSS.
.card {
  padding: var(--space-4);
  background: var(--color-surface);
  border-radius: var(--radius-md);
}

JSON / JavaScript

For JavaScript-based tools, linters, or CSS-in-JS environments that need raw values, you can import the generated JSON directly. 

import tokens from '../tokens.json';

// Usage in JS logic
const brandColor = tokens.color.brand.primary.value; // "var(--brand-500)"

// Usage in styling libraries
const Card = styled.div`
  background: ${tokens.color.surface.value};
  padding: ${tokens.space[4].value};
`;
